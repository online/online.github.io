Шпаргалка по использованию Git

Размещено 22.04.2019 Изменено 05.01.2019 Disqus:
558 1 мин.

В этом материале я буду собирать шпаргалки по использованию Git. Только то, что лично полезно мне. Может быть со временем данная информация окажется в закладках у кого-то из вас.

  • git init — создать новый репозиторий;
  • git add -A — добавить все файлы для отслеживания;
  • git commit -m "Добавил все файлы" — зафиксировать внесённые изменения с комментарием.

Парочка полезных картинок для понимания отличий между некоторыми командами (источник):

  • git add -A эквивалентна команде git add --all
  • git add -u эквивалентна команде git add --update
