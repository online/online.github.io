Управляем Hexo блогом на нескольких устройствах

Размещено 29.12.2018 Изменено 30.12.2018 Disqus:
816 1 мин.

После того как я переустановил и настроил Windows на стационарном ПК, движок Hexo почему-то не хотел полностью выполнять команду hexo g -d, которая отвечает за генерацию содержимого блога и публикацию изменений на GitHub Pages.

Решение для данной проблемы оказалось банально простым — нужно было установить официальный клиент GitHub для Windows , после чего я просто повторил вышеуказанную команду и во всплывающем окне у меня попросило логин и пароль от моего GitHub аккаунта.

Ещё осталось разобраться как быть с ситуацией, когда на одном компьютере необходимо управлять двумя блогами, которые находятся в двух разных GitHub аккаунтах. Пока удалось найти только такой способ, но его не пробовал.

Если вы читаете данную заметку и владеете ответом на последний вопрос — напишите, пожалуйста, в комментариях своё решение.
