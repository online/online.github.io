Hello, Hexo!

22.02.2017 | 28.05.2017 |

Для себя в Hexo открыл удобную фичу по вставке картинок в материалы. Оказывается, что достаточно в глобальном конфиге установить значение true для строки post_asset_folder, а затем воспользоваться командой hexo new post moy-super-post, что создаст в папке \source\_posts\ файл moy-super-post.md и папку moy-super-post, в которую можно поместить любые файлы, в том числе и изображения, а затем удобно ссылаться на них через встроенный в Hexo плагин:

За данную информацию хочу сказать спасибо официальному сайту Hexo, который, кстати, доступен на русском языке.
